# ПОРТ — Программа Организации Разнообразных Типов

## Описание проекта

Консольный файловый органайзер на Go. Сканирует указанную директорию и автоматически сортирует файлы по подкатегориям (Images, Documents, Music, Video, Archives) на основе расширения. Все операции записываются в лог-файл, по итогам сортировки выводится отчёт со статистикой по категориям.

## Возможности

- Сортировка файлов по 13 типам расширений: `.jpg`, `.jpeg`, `.png`, `.pdf`, `.doc`, `.docx`, `.txt`, `.mp3`, `.wav`, `.mp4`, `.avi`, `.zip`, `.rar`
- Автоматическое создание подкатегорий-директорий внутри рабочей директории
- Обработка коллизий имён через временную метку (`_2006-01-02_15-04-05`) перед расширением
- Логирование успешных операций и ошибок в `organizer.log`
- Подсчёт количества и суммарного размера файлов по каждой категории

## Требования

- Go 1.22 или новее
- macOS, Linux или Windows

## Установка

```bash
git clone https://github.com/raisultan/file-organizer.git
cd file-organizer
go build -o file-organizer
```

## Запуск

```bash
./file-organizer
```

Программа запросит путь к директории для сортировки. Нажмите Enter, чтобы использовать текущую директорию.

```
=== Файловый органайзер ===
Введите путь к директории для организации (Enter для текущей директории): ./test_files

=== Отчёт о перемещении файлов ===

Всего обработано файлов: 42
Общий размер: 128.50 KB

Статистика по категориям:

Documents:
  Файлов: 15, Размер: 45.30 KB

Images:
  Файлов: 20, Размер: 80.20 KB

Music:
  Файлов: 7, Размер: 3.00 KB

Лог операций: organizer.log
```
